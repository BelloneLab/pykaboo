# Python dependencies for PyKaboo (CamApp Live Detection)
# ---------------------------------------------------------------------------
# Quick start (CPU/GPU auto):
#     pip install -r requirements.txt
#
# For NVIDIA GPU acceleration (recommended — live mask+pose needs CUDA), install
# the CUDA build of PyTorch FIRST, then the rest:
#     pip install torch==2.12.0 torchvision==0.27.0 --index-url https://download.pytorch.org/whl/cu126
#     pip install -r requirements.txt
#
# External, NOT installed by pip:
#   * FFmpeg            — required for video recording. Install separately and
#                         put ffmpeg/ffprobe on PATH (e.g. `winget install Gyan.FFmpeg`).
#   * FLIR Spinnaker    — install the vendor PySpin wheel that ships in ./PySpin
#                         (matches the Spinnaker SDK + your Python version), e.g.
#                         pip install "PySpin/spinnaker_python-3.2.0.62-cp310-cp310-win_amd64/spinnaker_python-3.2.0.62-cp310-cp310-win_amd64.whl"
# ---------------------------------------------------------------------------
